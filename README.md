# 🚀 Space Blaster (Pygame Project)

## 🎮 Overview

**Space Blaster** is a 2D space shooting game developed using Python and Pygame.
The player controls a spaceship, destroys enemies, fights multiple bosses, and progresses through levels to win the game.

---


🎮 ✅ PLAY SYSTEM FEATURES
⏸️ LEFT/RIGHT Button for player movement
▶️ Keyboard SPACE button for shooting


## ✨ Features

* 🕹️ Smooth player movement and shooting system
* 👾 Enemy spawning with increasing difficulty
* 💀 Multiple boss battles (6 different bosses)
* 📈 Level progression system
* ❤️ Player lives system
* ⚡ Power-up system (health boost)
* 💥 Collision detection (player, enemy, bullets)
* 📊 Score & Highscore system
* 🧠 JSON-based data storage
* 🎵 Sound effects & background music
* 🎬 Start screen with player name input
* ❌ Game Over screen
* 🏆 Win screen after completing all levels

---

## 🧱 Technologies Used

## ▶️ How to Run the Game

### Step 1: Install Python

Download and install Python from: https://www.python.org/

### Step 2: Install Pygame

```bash
pip install pygame
```

### Step 3: Run the Game

```bash
python main.py
```
